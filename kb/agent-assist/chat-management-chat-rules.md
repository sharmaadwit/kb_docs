# Chat Management: Chat Rules

**Module**: Agent Assist

## What this feature does
Definition: Chat Rules are rules that prioritize and tag chats based on specific conditions, helping brands manage and categorize chats more effectively.

## Exact UI path
Agent Assist → Settings → Chat Management

## Prerequisites
- Access to **Agent Assist → Settings → Chat Management**.
- A test chat flow or team/agent setup to validate behavior.

## Setup path
- Navigate to "Settings" > "Chat Rules" in Agent Assist.

## Fields to configure
- channel is equal to WhatsApp.
- New Rule."
- tags to categorize chats based on different criteria

## Steps
1. Open Agent Assist.
2. Navigate to "Settings" > "Chat Rules" in Agent Assist.
3. Click **Save** to apply changes.
4. Click "Add New Rule.".
5. Choose from priority levels: High, Medium, and Low.
6. Add tags to categorize chats based on different criteria.
7. Click "Save" to save the rule.

## Validation / where to check
- Create or use a test chat and confirm the chat follows the expected routing/automation rule.

## Save / publish / deploy behavior
- Click **Save** (or **Save & Deploy** if available) to apply changes.

## Troubleshooting
- If the chat does not follow the expected rule, re-check rule order, conditions, and target team/agent configuration.

### Details
Section 1: Chat Rules

Definition: Chat Rules are rules that prioritize and tag chats based on specific conditions, helping brands manage and categorize chats more effectively.

Uses: Chat Rules streamline chat prioritization and tagging, improving chat organization and ensuring that chats receive the right level of attention and action.

Steps to Create Chat Rules:

Step 1: Accessing the Chat Rules Page

- Navigate to "Settings" > "Chat Rules" in Agent Assist.
Step 2: Create a New Rule

- Click "Add New Rule."
- Specify the name of the rule.
Step 3: Add Conditions to the Rule

- Define conditions that match the chat that needs to be tagged or prioritized, such as "channel is equal to WhatsApp."
Step 4: Specify Actions for the Rule

- Choose from priority levels: High, Medium, and Low.
- Add tags to categorize chats based on different criteria.
Step 5: Save the Rule

- Click "Save" to save the rule.
