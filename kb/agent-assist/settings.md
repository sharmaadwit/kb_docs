# Settings Overview

The Settings section acts as the central control hub for configuring and optimising your Agent Assist experience. From managing users and teams to automating workflows and configuring AI capabilities, this area allows administrators to align the platform with operational goals, compliance requirements, and team workflows.

## Configuration Areas

The Overview section provides a high-level introduction to the key configuration areas within Agent Assist. It helps you understand how different settings work together to shape your chat operations, team efficiency, and platform performance.

Perfect for:

- First-time administrators
- Understanding system controls

User Management enables comprehensive control over your team structure and access permissions.

Key Features:

- User Administration: Add, remove, and manage user accounts
- Team Creation: Organize agents into logical teams and departments
- Agent Status Configuration: Define availability statuses and working states
- Message Permissions: Control who can send, receive, and manage conversations
- Role Assignment: Assign appropriate access levels and permissions
- Feature Access Control: Manage which features each user can access
Benefits:

- Ensures smooth coordination across support and sales teams
- Maintains operational security and compliance
- Provides appropriate visibility levels for different roles
