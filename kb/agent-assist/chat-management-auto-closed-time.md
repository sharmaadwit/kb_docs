# Chat Management: Auto Closed time

**Module**: Agent Assist

## What this feature does
Section 1: Auto Closed Time

Common uses:

- Auto Closed Time ensures that chats are automatically closed from resolution after a specified period, helping maintain chat management efficiency and inactivity. By default the re-open time is configured as 0 which means the chats will be auto-closed instantly as and when chats are resolved

## Where to configure it
Agent Assist → Settings → Chat Management
