# Others: Chat Fields

**Module**: Agent Assist

## What this feature does
Chat Fields Chat fields are used to store qualitative values related to a chat. These fields can be created to categorize chats based on specific parameters, such as the type of issue being reported or the customer's preferred mode of communication. To create a chat field, go to the settings tab and click on "Chat Fields". Then select the type of field you want to create, such as dropdown, single line of text, or date time. Specify a name for the field and choose whether it should be required for agents to fill out.

## Where to configure it
Agent Assist → Settings → Others
