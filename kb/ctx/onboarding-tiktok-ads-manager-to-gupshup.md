# Pre-requisites

- A verified WABA (WhatsApp Business Account) number.
- Admin access to TikTok Ads Manager
- Access to Gupshup console
# Onboarding Ads Manager to Console

Step 1: Login to the Gupshup console panel

Step 2: In the left menu, click on "Click-to-chat ads" -> Ad Management

Step 3: Click on "Connect TikTok Ad Account"

Step 4: In the pop-up that opens, select all the assets that request permission and click on "Confirm"

Step 5: Once the user clicks the confirm button the user gets redirected to the console screen where the user selects the ad accounts

Step 6: Select the ad accounts by clicking on the check-box button and click on "Continue"

Once these steps are completed, the TikTok Ads Manager will be connected to the Gupshup console
