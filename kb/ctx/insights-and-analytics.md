## Step-by-step configuration
How to track performance of CTWA Campaigns and Ads

We have a dedicated analytics section for Click-to-chat ads on the Gupshup console. To access this, on the left hand menu, click on "Click-to-chat ads -> Analytics"

# Page 1: Ad Insights

This page gives detailed information regarding ad performance: Ad spends, lead funnel, retargeting metrics, CTR and impressions. Use the filters offered to filter campaigns/ads basis campaign name, ad name, date range.

# Page 2: Comparison

To compare performance of ads, head to this tab and select the ads you want to compare

# Page 3: Ad Performance

This report gives you a deeper comparative understanding on the CTWA ads are performing

# Page 4: Conversions API (CAPI)

This report showcases the number of Lead and Purchase signals that are sent to Meta. It shows both successful and failed signals.

# Page 5: Ad Attribution

This report shows the monetary revenue and non-monetary conversions that have been received from CTWA ads. Attribution API needs to be configured for data to be populated in this tab.
