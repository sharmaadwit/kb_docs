# Save Vs Save & Deploy

**Module**: Bot Studio

## Definition
- `Save` saves progress done so far in Bot Studio.
- `Save & Deploy` hosts the chatbot on a channel and pushes the saved details to live.

## Save
- Use `Save` while the journey is still being designed.
- The source page advises saving progress at regular intervals to avoid loss of effort.
- A click on `Save` performs a backend action and saves the entire progress.

## Save & Deploy
- Use `Save & Deploy` when the chatbot is ready to be seen in action.
- `Save & Deploy` allows the chatbot to be hosted on a channel.
- The deploy action pushes saved Bot Studio details to live.
- The source page states that `Save & Deploy` is mandatory to see changes in the live environment.
