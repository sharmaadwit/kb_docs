# Sending an Automated Campaign

Prerequisites:
- Active WABA in your project.
- Active source to receive events (Shopify or WhatsApp).

Steps:
1. Navigate to "Automated Campaigns".
2. Create a new campaign.
3. Select the trigger event (e.g., Cart Abandoned).
4. Configure the message template.
5. Set the delay if necessary.
6. Launch the campaign.
