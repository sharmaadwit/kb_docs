# Campaign Analytics

The campaign analytics is available for all the campaigns which are ‘sent’.
This page provides a preview of the campaign and high level analytics and in order to get to advanced stats, you can generate a response file that provides additional details regarding the campaign.

Following numbers are typically tracked:
- Total Sent: The number of messages successfully sent.
- Delivered: The number of messages delivered to the recipient's device.
- Read: The number of messages read by the recipients.
- Clicked: The number of clicks on links within the message.
- Failed: The number of messages that failed to deliver.
