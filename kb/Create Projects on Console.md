# Create Projects on Console

Gupshup's Console allows businesses to create projects to manage multiple departments or verticals.

Steps to create a new project:
1. Log in to the Console as Organisation Admin.
2. Click on the projects dropdown and select "Create new project".
3. Provide a project name and click "Create project".
4. You will be redirected to the new project.
5. Invite members and assign required modules.

Advantages:
- Allocation of modules to specific members.
- Centralised platform for managing messaging operations across departments.
