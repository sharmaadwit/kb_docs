# Workspace

The Workspace screen within the AI Admin module enables users to build, organize, and manage AI agents tailored to their specific business use cases.

Each workspace acts as a container for:
- Agent Personality
- Skills
- Tools
- Settings
- Supporting modules (Intents, Entities, Content, Teach, Evaluate)

Types of Workspaces:
- Generic: For broad use cases.
- Commerce: Tailored for retail and e-commerce journeys.
