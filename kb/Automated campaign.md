# Automated campaign

Introduction:
Automated Campaign is a feature available on Gupshup's Console product. The Goal of the feature is to support sending trigger based messages to users. For example, if a Shopify store is connected with Gupshup and a user abandons a cart, you can send a personalized message to nudge the user to complete the purchase.

Currently supported event sources:
- Shopify
- WhatsApp
