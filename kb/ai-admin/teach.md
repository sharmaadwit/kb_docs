# Teach

**Module**: Ai Admin

## Definition
- The utterances can be either added manually or through a predefined CSV file (sample file available to download) on the Teach tab
- The utterances with the identified Intents and utterances with the unidentified Intents are shown
- The Intents and Entities mapping can be changed
- “FAQ” and “Product Search, Q&A” Intent is introduced along with the Content Tags
