# Export and Import of Intents/Entities in AI Admin

The Intents and Entities can be exported and imported seamlessly from the Settings tab. It allows managing Intents and Entities, offering greater flexibility and convenience to users.

Sample: A sample ZIP file can be downloaded from the settings tab. This can be used to modify Intent/Entity and import in AI Admin.

Export: All the created Intents and Entities can be exported by clicking on the Export icon.

Import:- The ZIP file containing Intent and Entities can be imported by choosing a file >> import.

### Note:

- Only one ZIP file is imported.
- Once the Intent is created it cannot be deleted.
Description: The description of the workspace can be passed and applied to the overall workspace.

Precision Threshold: The precision threshold refers to the standard for accepting responses generated by the language model. Responses with a score falling between 10 and 70 on the accuracy scale are considered suitable, ensuring that only relevant and accurate suggestions are accepted. This threshold helps maintain the quality and accuracy of the responses provided by the model.

# Commerce Settings

- Show Out of Stock products: Users toggle to display/hide out-of-stock items. Defaulted to 'on,' it allows users to control product visibility based on stock availability.
- Apply Price Range: Enables users to search products within specified price brackets. If inactive, the bot won't accurately respond to price-specific queries.
- Apply Tags Filter: Users can filter products based on catalog ZIP file tags, offering customized product searches aligned with specific tags. For example:- "Show me red colour polo t-shirts". Here "red" colour is the tag. If set to inactive, the bot won't respond to tag-specific queries.
- Apply category filter: Enables product searches based on categories present in the catalog ZIP file, allowing users to find items within designated categories. If set to inactive, the bot won't be able to respond to category-specific queries. For example:- "Show me red colour t-shirts". Here "t-shirt" is the category.
# Developer Settings

- Content Continuity: Ensure seamless content flow as per the boolean value provided.
- URL Threshold: Set limits for URLs fetched as per the accuracy value.
- Q&A Threshold: Manage context depth for Q&A of content trained through the teach module.
- Fallback Configuration: Configure fallback messages for a few screened words in a pre-selected language.
- Prompt Editor: Prompt Editor will be provided to facilitate the creation, modification, and management of conversational prompts used by the chatbot.
- Workspace Configurator:- The workspace configurator is used to configure the message accuracy and description too. Moreover whatever configuration is being used here will be applied to the entire workspace too.
