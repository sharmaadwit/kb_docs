## Step-by-step configuration
Introduction: AI will be assisting users in writing effective Intent & Entity descriptions to improve runtime language understanding capabilities. AI enhanced description will be addressing following key issues observed in user written description are

- Lack of context
- Missing key information
- Use of vague language
Key Highlights:

- Enhance with AI button is introduced in Intent & Entity creation screen
- User need to provide base description (min 15 characters) to enable "Enhance with AI" feature
- User can rephrase the AI generated using rewrite button. Max 2 rewrites are allowed.
- User can edit the AI generated description by inserting it in the main intent/entity creation screen
