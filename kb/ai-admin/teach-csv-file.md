# Teach CSV File

**Module**: Ai Admin

## Definition
- For teaching an “FAQ” Intent in Generic Workspace and “Product Search, Q&A”; it is mandatory to have an existing Content Tag and recommended to have Answer and URL field
- For teaching a normal custom Intent created by the Bot Designer, it is recommended to have existing Entities
- Utterance and Intent are mandatory attribute
Updated 10 months ago
