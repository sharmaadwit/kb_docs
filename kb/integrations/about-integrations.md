# About integrations

**Module**: Integrations

## Definition
Gupshup supports the following partners, you can checkout the step-by-step process for each integration by clicking on the respective guide link on: https://console-docs.gupshup.io/docs

## Procedure
### Exact UI path
Gupshup Console → Integrations → About integrations

### Prerequisites
- Access to the target app/project in Gupshup Console.
- Any external endpoint/credential required by the integration.

### Fields to configure
- No explicit fields were identified in the source; use the UI controls shown on this page.

### Steps
1. Open Gupshup Console.
2. Go to **Integrations**.
3. Go to **About integrations**.
4. Click **Save** (or **Save & Deploy**) to apply changes.

### Validation / where to check
- Trigger a test event and confirm the target endpoint/system receives the callback or data.

### Troubleshooting
- If events do not arrive, verify the target URL/system is reachable and the configured event type matches the trigger.

### Save / publish / deploy behavior
- Click **Save** (or **Save & Deploy**) to apply changes.

### Setup path
- Go to **Integrations**.
- Go to **About integrations**.
