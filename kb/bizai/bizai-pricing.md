# BizAI for Partners Pricing

BizAI pricing is structured as a **per-token, usage-based model** with tiered base subscriptions.

## Pricing Tiers

### Agent Base Tier
**$50/month**

Includes:
- Up to 100,000 agent interactions per month
- 1 agent
- Basic knowledge management (up to 5 documents)
- Standard escalation rules
- Basic metrics dashboard

### Per-Token Overage
**$2.00 USD per 1 million tokens**

Interactions exceeding your base tier are charged at token rate.

**Estimating costs:**
- Typical interaction: 200-500 tokens
- 4-5 cents per interaction at overage rate
- 100k interactions = ~$200-500/month overage

Example:
- Base: 100k interactions/month
- Actual usage: 150k interactions/month
- Overage: 50k interactions × $0.00002/token ≈ $100

### Premium Features
**+$200/month**

Includes:
- Advanced knowledge management (unlimited documents, URLs, connectors)
- Custom agent instructions and behaviors
- Multi-language support optimization
- Priority support
- Custom integration assistance
- SLA guarantees

### Evaluation & Optimization Tools
**Included in Premium tier**

- Performance analytics dashboard
- Failure mode detection (automated analysis of escalation causes)
- Iteration recommendations (what to change in your knowledge base or prompts)
- A/B testing support
- Weekly reports and recommendations

## Multi-Channel Deployment

**No per-channel surcharge.**

One agent license covers:
- WhatsApp
- SMS
- Email
- Web widget

Deploy the same agent across all channels with zero additional cost.

## Resale & Partner Margins

**Partners can resell BizAI to their end customers.**

- Gupshup does **not** impose resale restrictions
- Partners are free to set their own pricing and packaging
- Example: Buy at $50/month base, resell at $99/month → $49/month margin
- No volume discounts or special arrangements required

## Cost Optimization Tips

1. **Start with base tier** — Test agent quality before scaling
2. **Optimize knowledge base** — Fewer, higher-quality documents reduce tokens
3. **Reduce escalations** — Better prompts and training → lower token usage
4. **Monitor metrics** — Weekly review of usage patterns
5. **Bundle channels** — Use same agent across WhatsApp, SMS, web to reduce total agents needed

## Billing & Invoicing

- **Monthly billing** — Charged on day of signup each month
- **Usage tracking** — Real-time dashboard shows current month's tokens
- **Overages billed next cycle** — Not charged in real-time
- **Annual discounts** — 15% off if paid annually (premium tier only)

## Frequently Asked Questions

**Q: What happens if I exceed my base tier?**
A: Interactions are charged at the per-token rate. No service interruption.

**Q: Can I downgrade from Premium?**
A: Yes, anytime. Refund prorated for remainder of month.

**Q: Do template messages cost extra?**
A: No. BizAI charges only for agent interactions, not templated messages.

**Q: Can I use BizAI across multiple WEbApps?**
A: No. One agent per license. Multiple agents require multiple subscriptions.
