# Backward Compatibility for existing AI Journeys

### Generic Workspace Migration
Data Workspaces and associated content would move to individual newly created Workspaces. The name of the Workspace would be the respective Data Workspace name. The system would create an intent “FAQ” in these workspaces.

### Intent and Entity Migration
Existing intents and entities will be migrated to the new workspace structure to ensure continued functionality of AI journeys.

### Commerce Workspace Migration
Specific migration steps are in place for commerce-related workspaces to ensure product search and query features continue to work as expected.
