# Manage Skills

**Module**: SuperAgent

> [!NOTE]
> **Prerequisites · ~2 minutes**
> You need a SuperAgent account. To create custom skills, enable the **Build Skills & Recipes** toggle in the chat input bar.

Skills add capabilities to SuperAgent—things like creating charts, sending emails, searching the web, and managing tasks. Here's how to browse and manage them.

## See it in the app

Same walkthrough as the in-app Skills Guide: the step buttons below match the numbered steps that follow.

### Step 1: Turn on Build Skills & Recipes in the input bar

In the chat input bar, turn on **Build Skills & Recipes** so the AI can create and edit skills for you.

### Step 2: Ask the AI in chat to create a skill

Type what you want in the chat box—for example, ask for a new skill that calls a public API—then send your message.

### Step 3: Watch the AI build the skill files

The AI creates the skill folder and files (e.g. `SKILL.md`, `action.py`) and confirms when it's done.

### Step 4: Open the Skills modal to browse and enable

Open **Skills** from the sidebar to launch **Manage Skills**. Search, filter, and use the toggles to enable or disable skills for your account.

---

## Open the Skills panel

Click the Skills icon in the left sidebar to open the Skills panel. You'll see a list of all available skills, organized by category.

## Browse by category

Skills are grouped into categories such as Productivity, Data, Communication, and more. Use the category tabs or filters to find skills that match what you need. Each skill shows a short description of what it does.

## Enable and disable skills

Toggle a skill on or off with the switch next to it. When enabled, the AI can use that skill in your conversations. When disabled, it won't appear in the AI's toolkit. You can change this anytime.

## Search for skills

Use the search box at the top of the Skills panel to find skills by name or keyword. For example, search for "chart" to find chart-related skills or "email" for email-related ones.

## Request access

Some skills may be restricted or require approval. If you see a skill you can't use, look for a "Request access" or "Contact admin" option. Your administrator can grant access if appropriate.

## Import and export skills

If your organization uses custom skills, you may be able to import them from a file or link. Similarly, you can export your skill configuration to share with others or back up your setup. Look for Import/Export options in the Skills panel menu.

> [!TIP]
> **Start simple**
> Enable a few skills at a time and try them in conversation. Add more as you discover what you need.
