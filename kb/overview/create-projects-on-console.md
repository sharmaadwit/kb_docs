# Create Projects on Console

As businesses grow and expand their messaging operations, managing multiple departments or verticals can become a daunting task. Gupshup's Console offers a powerful set of features to simplify messaging operations, including the ability to create projects. Creating a project on Console is a simple process that can help businesses streamline their messaging operations and manage multiple departments or verticals efficiently.

Here are the steps to create a new project on the Console:

- Log in to the Console as the Organisation Admin using this link.
- Click on the projects dropdown menu and select Create new project.
- Provide a name for the project and click on the Create project button.
- Once the project is created, You will be automatically redirected to the new project.
- Invite members who need to access the project and assign the required modules to them. The Project Admin can assign one or more modules to members of the project on a need basis.
By following these simple steps, businesses can create and manage projects on Console efficiently. The allocation of resources and modules can be done seamlessly, making it a valuable tool for businesses that manage multiple departments or verticals.

One of the key advantages of using the Console's project feature is the ability to allocate modules to specific members of the project. The Project Admin can assign modules to members on a need basis, ensuring that each member has access to the tools and resources they require to perform their tasks efficiently.

In addition, Console's project feature offers a centralised platform to manage messaging operations across multiple departments or verticals. This enables businesses to optimize their operations and achieve better results.

Console's project feature is just one of the many powerful tools available to businesses using Gupshup's messaging solutions. If you have any queries or face any issues while creating a project on Console, you can email console-support@gupshup.io. The Gupshup team will be happy to answer your questions and provide you with any assistance you need.
