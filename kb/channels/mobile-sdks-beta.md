# Mobile SDKs (Beta)

- You can now integrate a chat widget in your mobile apps for Android and iOS using the Mobile SDKs for the Web channel.
- All Web channel capabilities, including features and message types, are available in the chat widget integrated through the SDKs.
## SDK Details

- Android Native and iOS Native SDKs are available for the integration.
- A bridge connector for React Native (mobile application development framework) available for Native SDKs. Support for other mobile application development frameworks such as Flutter, Cordova, Xamarin and more will be available shortly.
- Support for other mobile application development frameworks such as Flutter, Cordova, Xamarin and more will be available shortly.
## Supported Operating Systems & Versions

- Android 9 and above
- iOS 13 and above
