## Setting the Persistent Menu

- Go to the Settings in the Web channel.
- Go to the Persistent Menu tab.
- Click Add Menu Button and select the button type from the dropdown. For Reply button, enter the button text. For URL button, enter the button text and the URL you wish to redirect to.
- For Reply button, enter the button text.
- For URL button, enter the button text and the URL you wish to redirect to.
- Click Save. The Persistent Menu will start appearing for all users in the linked Instagram account's DM.
## Deleting the Persistent Menu

- If you wish to delete any menu button, click the delete icon next to it and click Save.
- If you wish to delete all buttons, click Delete All at the bottom left corner of the tab.
