The recipe in action
Use case 1: Abandoned cart campaign
Check if the customer is using Shopify. If yes, check if the shopify integration is done, if not call shopify integration skill.
Then list all shopify events, call automated campaign skill to create an abandoned cart campaign
Execute and make the campaign live if the user gives a go ahead. 
Use case 2: COD management
Check if the customer is using Shopify. If yes, check if the shopify integration is done, if not call shopify integration skill.
Create an interactive automated campaign which triggers based on order placed event where payment type is COD. The template message should ask the user about order confirmation and should have 2 buttons “Yes” and “No”. In the interactive part of this campaign, call Shopify API to confirm the order when the user clicks “Yes”. 
Use case 3: Configuring sales assistant for FAQ/product recommendations
Call AI Admin skill to set up a FAQ agent.
Use case 4: Configuring sales assistant for driving sales
Call AI Admin skill to set up a FAQ agent.
Use case 5: Configuring sales assistant for post purchase support
Call AI Admin skill to set up a FAQ agent.

Check if the customer is using Shopify. If yes, check if the shopify integration is done, if not call shopify integration skill.

Execute and make the campaigns live if the user gives a go ahead.
